# Análisis de diversidad genética
La diversidad genética dentro de individuos y poblaciones proporciona información clave sobre:
- Historia demográfica
- Tamaño efectivo poblacional
- Endogamia
- Flujo génico
- Estructura genética

## 1. Heterocigosidad global
La heterocigosidad individual es la relación entre el número de sitios heterocigotos respecto del número total de sitios evaluados. Para datos de NGS, estimamos esta cantidad mediante el Espectro de Frecuencia de Sitios (SFS) individual.
```bash
mkdir 5-diversidad
cd 5-diversidad
cp /data/courses/course2026/course_files/scripts/diversidad/*.sh .
```
### 1.1 Estimación del SFS individual
Usando ANGSD, estimamos probabilidades de genotipo a nivel individual, a partir del cual obtendremos una estimación del SFS. Creamos el script de la tarea (cambiar USERNAME por el nombre de grupo):
```bash
#!/bin/bash

# Nombre del proceso:
#SBATCH --job-name=1-angsd_saf

# Proyecto:
#SBATCH --account=tipo-iii
#SBATCH --partition=standard # Nombre de la partición

# Recursos:
#SBATCH --time=24:00:00 # Límite de tiempo de corrida
#SBATCH --cpus-per-task=15 # Límite de CPUs para cada tarea en tareas de múltiples instancias
#SBATCH --mem=25G # Límite de memoria

# Archivos de salida
#SBATCH --output=/data/courses/course2026/USERNAME/logs/angsd_saf_o%j # Nombre del archivo de salida stdout
#SBATCH --error=/data/courses/course2026/USERNAME/logs/angsd_saf_e%j # Nombre del archivo de error stderr

# Ambiente de trabajo
module load angsd
module list

# Mensaje informativo
echo "Tarea ${SLURM_JOB_ID} name ${SLURM_JOB_NAME} iniciado en:"
date
echo "Nodo:"
hostname

# Comandos a ejecutar
ref_genome="/data/courses/course2026/course_files/mapeo_variantes/ppa_v1.asm.fasta"
input_file="/data/courses/course2026/course_files/diversidad_genetica/HUSM36.Neltuma_pallida.bam"
input_prefix=$(basename $input_file .bam)
threads=10

mkdir -p Heterozygosities

echo "Estimar las probabilidades de genotipo para una muestra"
angsd -GL 2 -doMajorMinor 1 -doGlf 2 -doSaf 1 \
 -doCounts 1 -nThreads $threads \
 -minMapQ 30 -minQ 20 -setminDepthInd 2 \
 -remove_bads 1 -uniqueOnly 1 -baq 1 -C 50 \
 -ref $ref_genome -anc $ref_genome -i $input_file \
 -out Heterozygosities/${SLURM_JOB_ID}.${input_prefix}

date
echo "Estimar el espectro de frecuencia de sitios"
/opt/ohpc/pub/apps/angsd/angsd/misc/realSFS Heterozygosities/${SLURM_JOB_ID}.${input_prefix}.saf.idx \
  -fold 1 > Heterozygosities/${SLURM_JOB_ID}.${input_prefix}.est.ml

echo "Job finalizado en:"
date
```
Someter el proceso al organizador:
```bash
sbatch 1-angsd_saf.sh
```
El archivo `est.ml` contiene las probabilidades en orden:
```
p(AA)  p(Aa)  p(aa)
```
Donde:
- AA = homocigoto para el alelo mayor
- Aa = heterocigoto
- aa = homocigoto para el alelo menor

### 1.2 Estimación de la heterocigosidad individual
La heterocigosidad corresponde a la frecuencia de heterocigotos:
```bash
awk '{print $2 / ($1 + $2) * 100}' Heterozygosities/file.est.ml
```

## 2. Heterocigosidad en ventanas genómicas
La heterocigosidad no es uniforme a lo largo del henoma. Regiones con baja heterocigosidad pueden indicar:
- Selección reciente
- ROH
- Baja recombinación

### 2.1 Estimación de theta por sitio
La heterocigosidad a nivel poblacional se encapsula en los estimados de theta. Podemos obtener un estimado global y en ventanas de la siguiente manera. Recordar cambiar los USERNAME por los nombres de grupo:
```bash
#!/bin/bash

# Nombre del proceso:
#SBATCH --job-name=2-angsd_theta

# Proyecto:
#SBATCH --account=tipo-iii
#SBATCH --partition=standard # Nombre de la partición

# Recursos:
#SBATCH --time=24:00:00 # Límite de tiempo de corrida
#SBATCH --cpus-per-task=15 # Límite de CPUs para cada tarea en tareas de múltiples instancias
#SBATCH --mem=25G # Límite de memoria

# Archivos de salida
#SBATCH --output=/data/courses/course2026/USERNAME/logs/angsd_theta_o%j # Nombre del archivo de salida stdout
#SBATCH --error=/data/courses/course2026/USERNAME/logs/angsd_theta_e%j # Nombre del archivo de error stderr

# Ambiente de trabajo
module load angsd
module list

# Mensaje informativo
echo "Tarea ${SLURM_JOB_ID} name ${SLURM_JOB_NAME} iniciado en:"
date
echo "Nodo:"
hostname

# Comandos a ejecutar
angsd_prefix="HUSM_NORTH"
input_saf="/data/courses/course2026/course_files/diversidad_genetica/HUSM_NORTH.saf.idx"
threads=10

echo "Estimar el espectro de frecuencia de sitios a nivel poblacional"
/opt/ohpc/pub/apps/angsd/angsd/misc/realSFS $input_saf -P $threads -fold 1 -seed 1 > ${SLURM_JOB_ID}.${angsd_prefix}.sfs

echo "Estimar indices de diversidad globales para cada segmento"
/opt/ohpc/pub/apps/angsd/angsd/misc/realSFS saf2theta $input_saf -outname ${SLURM_JOB_ID}.${angsd_prefix}.saf2theta \
  -sfs ${SLURM_JOB_ID}.${angsd_prefix}.sfs -fold 1 -P $threads -seed 1

/opt/ohpc/pub/apps/angsd/angsd/misc/thetaStat do_stat ${SLURM_JOB_ID}.${angsd_prefix}.saf2theta.thetas.idx \
  -outnames ${SLURM_JOB_ID}.${angsd_prefix}.saf2theta.thetas.idx.chrs

echo "Job finalizado en:"
date
```
Luego, someter la tarea:
```bash
sbatch 2-angsd_theta.sh
```
Los resultados incluyen el `sfs` y los índices de diversidad genética de cada segmento:
```bash
seq 1 100 | tr -s '\n' '\t' | sed 's/$/\n/' | cat - HUSM_NORTH.sfs | column -t | less -S

column -t HUSM_NORTH.saf2theta.thetas.idx.chrs.pestPG | less -S
```

### 2.2 Estimación en ventanas
Calculamos los índices de diversidad nucleotídica y theta de Watterson en ventanas de 100 kb con pasos de 10kb. Cambiar los USERNAME por el nombre de grupo:
```bash

#SBATCH --job-name=3-angsd_window

# Proyecto:
#SBATCH --account=tipo-iii
#SBATCH --partition=standard # Nombre de la partición

#SBATCH --error=/data/courses/course2026/USERNAME/logs/angsd_window_e%j # Nombre del archivo de error stderr

# Ambiente de trabajo
module load angsd
module list

# Mensaje informativo
echo "Tarea ${SLURM_JOB_ID} name ${SLURM_JOB_NAME} iniciado en:"
date
echo "Nodo:"
hostname

# Comandos a ejecutar
angsd_prefix="HUSM_NORTH"
input_saf="/data/courses/course2026/course_files/diversidad_genetica/HUSM_NORTH.saf.idx"
threads=10
window=50000
step=10000


echo "Estimar el espectro de frecuencia de sitios a nivel poblacional"
/opt/ohpc/pub/apps/angsd/angsd/misc/realSFS $input_saf -P $threads -fold 1 -ssed 1 > ${SLURM_JOB_ID}.${angsd_prefix}.sfs

echo "Estimar indices de diversidad globales para cada segmento"
/opt/ohpc/pub/apps/angsd/angsd/misc/realSFS saf2theta $input_saf -outname ${SLURM_JOB_ID}.${angsd_prefix}.saf2theta \

  -win $window -step $step -outnames${SLURM_JOB_ID}.${angsd_prefix}.saf2theta.thetas.idx.w$window.s$step

```

## 3. Densidad de variantes
La densidad de SNPs es un indicador directo de diversidad genética. Para esto, primero obtenemos las posiciones a partir del archivo `beagle`:
```bash
zcat file.beagle.gz | sed '1d' | cut -f1 | sed 's/_/\t/' > SNP_positions.txt
```
Luego, convertimos a formato BED:
```bash
awk '{print $1 "\t" $2-1 "\t" $2}' SNP_positions.txt > SNP_positions.bed
```

## 4. Relaciones de parentesco
Usando las probabilidades de genotipo, estimaremos coeficientes de parentesco entre individuos. Esto permite detectar relaciones biológicas o artificiales como:
- Hermanos, padres-hijos
- Muestras duplicadas
- Individuos no relacionados

### 4.1 Ejecutar NGSrelate
Primero, creamos un script con el software y usando el archivo `beagle` filtrado. Cambiar los USERNAME por el nombre de grupo:
```bash
#!/bin/bash

# Nombre del proceso:
#SBATCH --job-name=5-relatedness

# Proyecto:
#SBATCH --account=tipo-iii
#SBATCH --partition=standard # Nombre de la partición

# Recursos:
#SBATCH --time=24:00:00 # Límite de tiempo de corrida
#SBATCH --cpus-per-task=15 # Límite de CPUs para cada tarea en tareas de múltiples instancias
#SBATCH --mem=25G # Límite de memoria

# Archivos de salida
#SBATCH --output=/data/courses/course2026/USERNAME/logs/relatedness_o%j # Nombre del archivo de salida stdout
#SBATCH --error=/data/courses/course2026/USERNAME/logs/relatedness_e%j # Nombre del archivo de error stderr

# Ambiente de trabajo
module load ngsrelate
module load angsd
module list

# Mensaje informativo
echo "Tarea ${SLURM_JOB_ID} name ${SLURM_JOB_NAME} iniciado en:"
date
echo "Nodo:"
hostname

# Comandos a ejecutar
angsd_prefix="HUSM_NORTH"
bam_list="bam_file.list"
ref_genome="/data/courses/course2026/course_files/mapeo_variantes/ppa_v1.asm.fasta"
threads=10
nind=$(wc -l $bam_list | cut -d' ' -f1)
let minind=$nind/2

echo "Estimar las probabilidades de genotipo para una muestra"
angsd -b $bam_list -GL 2 -doMajorMinor 1 \
 -doGlf 3 -doPlink 2 -doPost 1 -doCounts 1 \
 -nThreads $threads -SNP_pval 1e-6 -doGeno -1 \
 -doMaf 3 -minMapQ 30 -minQ 20 -minMaf 0.05 \
 -geno_minDepth 2 -setMinDepthInd 2 -minInd $minind \
 -remove_bads 1 -uniqueOnly 1 -postCutoff 0.95 -baq 1 -C 50 \
 -ref $ref_genome -anc $ref_genome -out ${SLURM_JOB_ID}.$angsd_prefix

echo "Construir el archivo de entrada a partir de las frecuencias alelicas de angsd"
zcat ${SLURM_JOB_ID}.${angsd_prefix}.mafs.gz | cut -f7 | sed 1d > ${SLURM_JOB_ID}.${angsd_prefix}.freq

echo "Inferir parametros de parentesco y endogamia"
ngsRelate -g ${SLURM_JOB_ID}.${angsd_prefix}.glf.gz -n $nind -f ${SLURM_JOB_ID}.${angsd_prefix}.freq -O ${SLURM_JOB_ID}.${angsd_prefix}.newres

echo "Job finalizado en:"
date
```
Sometemos el proceso al organizador:
```bash
sbatch 5-relatedness.sh
```
Obtenemos varios valores, los cuales pueden visualizarse preliminarmente:
```bash
head -n 100 file.newres | column -t | less -S
```
Entre estos, el coeficiente de parentesco nos permite tener una idea de la relación entre individuos:

| Coeficiente de parentesco | Relación|
|----------------------------|--------|
| 0.5  |  Mismo individuo o gemelos idénticos|
| 0.25 |  Relación padre-hijo|
| 0.25 |  Hermanos completos|
| 0.125 | Relación abuelo-nieto|
| 0.125 | Medio hermanos|
| 0.0625 | Primos de primer grado|

```bash
cut -f19 file.newres | sort | uniq -c
```

### 4.2 Coeficiente de endogamia
Por otro lado, obtenemos una estimación del nivel de endogamia `F` para cada individuo. Este es la probabilidad de que un individuo posee dos alelos en un locus que son idénticos por descendencia (IBD) de un ancestro en común.

El coeficiente de endogamia va de 0 a 1 y mide la homocigosidad en un individuo debido al parentesco entre los padres. Mientras el valor de F es más alto, indica una mayor homocigosidad, lo cual puede llevar a depresión endogámica.

```bash
cut -f14 file.newres | sort | uniq -c
```
